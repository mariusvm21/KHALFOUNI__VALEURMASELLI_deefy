Tableau de bord des fonctionnalités

Fonctionnalité : Mes playlists
Statut : Réalisée
Description : Affiche toutes les playlists de l’utilisateur connecté.

Fonctionnalité : Afficher playlist
Statut : Réalisée
Description : Affiche la playlist sélectionnée et la stocke en session. Accessible uniquement au propriétaire ou à l’admin.

Fonctionnalité : Ajouter une piste
Statut : Réalisée
Description : Permet d’ajouter une piste à la playlist courante via un formulaire. Accessible uniquement au propriétaire.

Fonctionnalité : Créer une playlist vide
Statut : Réalisée
Description : Permet de créer une nouvelle playlist vide. Devient la playlist courante et est stockée en base. Accessible uniquement à un utilisateur authentifié.

Fonctionnalité : Afficher la playlist courante
Statut : Réalisée
Description : Affiche la playlist actuellement stockée en session.

Fonctionnalité : S’inscrire
Statut : Réalisée
Description : Création d’un compte utilisateur avec le rôle STANDARD. Vérification du format email et longueur du mot de passe.

Fonctionnalité : S’authentifier
Statut : Réalisée
Description : Authentification d’un utilisateur existant. Mot de passe stocké de façon sécurisée (hashé).
